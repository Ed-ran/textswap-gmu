# User Stories — TextSwap

## US-SELL-001: List & Sell Safely
As a student seller, I want to list a used textbook and get paid through escrow after the buyer confirms pickup/delivery, so that I can sell safely without being ghosted.

**Acceptance Criteria**
- .edu account + verification
- Create listing (title, condition, course, price, photos) in ≤2 min
- Buyer payment held in escrow
- Buyer confirm → escrow releases payout
- Dispute path with short response window

## US-BUY-001: Buy with Confidence
As a buyer, I want my funds held until I receive the book, so that I don’t risk paying for nothing.

**Acceptance Criteria**
- Browse/search by title/course
- Pay into escrow and see order status
- Confirm receipt to release funds
- Flag a dispute within window
