# User Workflow — Seller

1) Sign up with .edu → verify
2) New Listing → add title, course, condition, price, photos
3) Publish → visible to nearby campus students
4) Buyer taps “Buy” → payment goes to escrow
5) Arrange pickup on campus (or ship) via in-app messages
6) Buyer confirms receipt → escrow releases payout to seller
7) Rate transaction; dispute window closes
