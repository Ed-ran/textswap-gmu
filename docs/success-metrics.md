# Success Metrics — Pilot Targets

**Experience & Trust**
- Completion rate ≥ 80% within 72 hours of escrow
- Dispute rate ≤ 3%; average resolution ≤ 48 hours
- Median time-to-sale ≤ 7 days

**Growth & Liquidity**
- 200+ active listings in 30 days (GMU)
- Buyer conversion (view → buy) ≥ 8%
- Repeat users (60-day) ≥ 25%

**Satisfaction**
- Seller NPS ≥ +30; Buyer NPS ≥ +30
- “Felt safe using TextSwap” ≥ 85% agree
